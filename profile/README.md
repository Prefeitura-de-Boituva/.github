## Sobre

Este repositório reúne os projetos de desenvolvimento de software, engenharia de dados, automações, microsserviços, pipelines de integração, APIs e demais iniciativas tecnológicas desenvolvidas pela Prefeitura de Boituva.

Nosso objetivo é promover uma administração pública mais moderna, eficiente e orientada por dados, utilizando tecnologia para aprimorar a gestão municipal e a prestação de serviços ao cidadão.

---

## Missão

Desenvolver soluções tecnológicas que fortaleçam a transformação digital da Prefeitura de Boituva, promovendo inovação, integração entre sistemas, qualidade da informação e melhoria contínua dos serviços públicos.

## Contribuição

Os projetos deste repositório são destinados ao desenvolvimento interno da Prefeitura de Boituva.

Antes de contribuir:

1. Crie uma branch para sua funcionalidade;
2. Utilize commits descritivos;
3. Atualize a documentação quando necessário;
4. Abra um Pull Request para revisão.
